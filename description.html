Introduction

A Light Weight App that takes appointment and store into the file and search too. This app Understand Concept making of Console Project. 

Description

This Sample will teach you making of Console Projects in C# with using of File I/O.  This App have every single feature like add appointment, view appointment, search etc... This app contains background changing coding  and File System coding. This app automatically makes directories in your system and deleting appointment file in your system without any problem. It easy to handle and more feature addons.

 

This is Welcome page of Appointment Application. A easy interface which say's Everything about Task. Any One can run this app with their easy dashboard.

Adding Apointment page will help you to add appointment. This page have autocorrect option which correct your date. Only Time need to add manually with AM/PM. Enter Note option is not compulsary you can skip through enter button.

View Appointment will View your appointment Easily Just you have to enter only filename exactly without any extension then you can view your appointment. if you want too continue viewing the enter 'Y' or 'N' for back to dashboard.

This search feature is good. We can search  by Year. if You Entering 2015 then it show data related 2015 as shown in figure.

Deleting files feature will help you to delete file seperately by just typing filename exactly as per shown as same without any extension. This will notify you whether your file deletion is successfull or not.

This Clearing Wizard will delete your all Files except Directory. this option will clearing all your files which stored in appointment Folder.

Deleting Directoy will totally delete your file or folder as Appointment app created when you created your first appointment. this will Warn you whether you want to delete file or not. Remember this will delete your Entire directly of Your Appointment Foder. it is safe.

 

This About window will show the Programmer name's, Email and Version as per above window shown.
